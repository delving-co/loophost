Welcome to loophost.

WHAT:
Turnkey localhost management for students, freelancers, agencies and development teams of all sizes.
WHY:
Because it's 2023, and friends don't let friends run without SSL.
HOW:
Claim a username with your GitHub credentials, and we'll do the rest.
Specifically, we'll generate a private key (that never leaves your computer) and set up DNS records and SSL certificates for *.<github-username>.fling.dev.
When you visit https://yourapp.username.fling.dev, you'll be able to choose any locally running web app (in any language, and on any port or web socket) to bind to that name.
Read on for more of the technical details.
